# Polar Gini Curve (PGC) Technique for Biomarker Detection in Single-cell Data

## Overview

This repository contains the Python implementation of the Polar Gini Curve (PGC) technique, originally written in MATLAB. It's an innovative research method designed for the detection of biomarkers from single-cell data, incorporating gene expression and 2D embedded visual information.

## Background

The PGC technique proposes that if the PGC of a specific gene's cell-points is similar to the PGC of all cell-points in a cluster, that gene is likely a cluster marker. This method has been validated on several simulation case studies.

In a practical demonstration using neonatal mouse heart single-cell data, the technique helped detect potential biomarkers and identify novel subtypes of cardiac muscle cells.

## Repository Contents

The repository contains:

1. **Source Code:** The Python implementation of the PGC technique, adapted from the original MATLAB script.
2. **Datasets:** Diverse datasets, including the neonatal mouse heart single-cell data.
3. **Results:** Outcomes from the various simulation case studies.
4. **Documentation:** In-depth documentation on how to utilize the code and interpret the results.

## Usage

Researchers and bioinformatics professionals can use the Polar Gini Curve (PGC) technique to study biomarkers in single-cell data. This method offers a distinctive approach to understand gene expression and visual information to discover potential biomarkers in a cell cluster.

To use this tool:

1. Clone or download this repository.
2. Follow the instructions outlined in the documentation for installation and usage.
3. Use the included datasets for testing and validation.
4. For custom datasets, ensure they conform to the input format outlined in the documentation.
